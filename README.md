# Super-MIKU-Ball

Super-MIKU-Ball es un videojuego inspirado en Super Monkey Ball, pero con un giro divertido: el personaje principal es Miku, quien se encuentra dentro de una bola. El objetivo del juego es recolectar puerros, Rei Chikitas, y Teto Peras en dos niveles diferentes, mientras se compite contra el tiempo para llegar a la meta, que es una puerta Torii en alguna parte del nivel. Los jugadores reciben puntos según el tiempo restante al completar el nivel y los objetos recolectados.

La banda sonora incluye temas de Shibayan Records y música del género bossa nova, creando una atmósfera única y entretenida.

Este proyecto fue desarrollado en Unity como parte de la materia Computación Gráfica en EAFIT, por Esteban Vergara Giraldo y Samuel Lopera.
